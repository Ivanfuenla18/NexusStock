# 🚀 NexusStock - Fullstack Inventory Management

Sistema profesional de gestión de inventarios construido con el stack **MERN** (MongoDB, Express, React, Node.js).

## 🛠️ Tecnologías utilizadas

- **Backend:** Node.js & Express
- **Base de Datos:** MongoDB Atlas (Mongoose)
- **Seguridad:** Bcryptjs para encriptación de contraseñas y JWT (próximamente)
- **Frontend:** React (en desarrollo)

## 📂 Estructura del Proyecto

```text
NexusStock/
├── Server/          # API Restful con Node.js
│   ├── config/      # Conexión a DB
│   ├── controllers/ # Lógica de negocio
│   ├── models/      # Esquemas de Mongoose
│   └── routes/      # Endpoints de la API
└── Client/          # Interfaz de usuario (React)
```
